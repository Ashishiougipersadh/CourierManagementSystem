# Courier Management System

A web-based driver management system built with ASP.NET Core MVC, designed to streamline the management of drivers, deliveries, and related operational data.

---

## Project Overview

The Courier Management System is an ASP.NET Core MVC application that allows administrators to manage drivers and their delivery-related information efficiently. The system focuses on simplifying driver record management, improving data organisation, and providing a structured way to track driver details within a delivery workflow.

Administrators can create, update, view, and delete driver records, ensuring that all driver information is kept up to date and easily accessible.

---

## Key Features

### Driver Management
- Add new drivers to the system  
- View all existing drivers  
- Update driver details  
- Delete driver records  

### System Features
- ASP.NET Core MVC architecture  
- Entity Framework Core integration  
- SQL Server database support  
- Clean and scalable project structure
